# 📋 نظام التصحيح الآلي لبطاقات الإجابة

نظام ويب متقدم لتصميم وتصحيح بطاقات الإجابة بالكاميرا أو الماسح الضوئي باستخدام الذكاء الاصطناعي.

## ✨ الميزات الرئيسية

### 🎓 إدارة الفصول والطلاب
- ✅ إضافة فصول بسهولة
- ✅ تسجيل الطلاب يدوياً أو بدفعات
- ✅ استيراد من ملفات Excel/CSV
- ✅ إدارة سجل الحضور والدرجات

### 📝 تصميم البطاقات
- ✅ ثلاث أشكال: ورقة كاملة، بطاقات ٤ في الصفحة، أسئلة + إجابة
- ✅ دعم أنواع أسئلة: اختيار من متعدد، صواب/خطأ، مزاوجة
- ✅ ترقيم وترتيب تلقائي
- ✅ باركود QR لكل طالب (في وضع الاستيراد)
- ✅ معاينة فورية

### 📷 التصحيح الآلي
- ✅ رفع صور من الكاميرا أو الماسح الضوئي
- ✅ كشف تلقائي للأركان السوداء
- ✅ معالجة ذكية للصور بـ OpenCV.js
- ✅ قراءة الباركود QR
- ✅ تحليل التظليل بدقة عالية
- ✅ مسح مباشر متعدد (Live Scan)

### 📊 النتائج والتقارير
- ✅ عرض النتائج في جداول
- ✅ تصدير إلى Excel/CSV
- ✅ سجل أعمال تراكمي حسب الصف
- ✅ حفظ سحابي بـ Firebase (اختياري)

## 🚀 البدء السريع

### المتطلبات
- متصفح حديث يدعم:
  - HTML5 Canvas
  - File API
  - getUserMedia (للكاميرا)
  - WebGL (لـ OpenCV.js)

### التثبيت

#### 1️⃣ نسخ الملفات
```bash
git clone https://github.com/manasik-hotel/classroom-system.git
cd classroom-system
```

#### 2️⃣ إعداد Firebase (اختياري)

إذا أردت حفظ سحابي:

**أ) إنشء مشروع Firebase:**
1. اذهب إلى [Firebase Console](https://console.firebase.google.com)
2. اضغط "New Project"
3. اتبع الخطوات

**ب) نسخ الإعدادات:**
```bash
# انسخ الملف النموذجي
cp config.example.js config.js
```

**ج) تعديل config.js:**
```javascript
const FIREBASE_CONFIG = {
  apiKey: "YOUR_API_KEY",
  authDomain: "your-project.firebaseapp.com",
  projectId: "your-project-id",
  storageBucket: "your-project.appspot.com",
  messagingSenderId: "YOUR_SENDER_ID",
  appId: "YOUR_APP_ID"
};
```

> ⚠️ **هام:** لا تشارك `config.js` على GitHub. استخدم `.gitignore`

#### 3️⃣ فتح الملف
```bash
# الطريقة 1: استخدم خادم محلي
python3 -m http.server 8000
# ثم افتح: http://localhost:8000/index-improved.html

# الطريقة 2: افتح مباشرة في المتصفح
# انقر مزدوجاً على: index-improved.html
```

## 📖 دليل الاستخدام

### 1️⃣ إضافة الفصول

**التبويب: 👥 الطلاب والفصول**

```
اسم الصف: أول ثانوي / 1
رمز الصف: 101
→ اضغط: ➕ إضافة الصف
```

### 2️⃣ تسجيل الطلاب

**الخيار أ: يدويّ**
```
اسم الطالب: محمد نجيب حكمي
رقم الهوية: 1061532883
الصف: أول ثانوي / 1
→ اضغط: ➕ إضافة الطالب
```

**الخيار ب: من Excel**
```
الصف: أول ثانوي / 1
→ اختر ملف Excel (عمود A: الهوية، عمود B: الاسم)
→ اضغط: ⬆️ استيراد
```

### 3️⃣ تصميم البطاقة

**التبويب: 🖨️ التصميم**

```
1. أدخل بيانات الاختبار:
   - عنوان الاختبار
   - الإدارة التعليمية
   - المدرسة
   - المادة والصف والفترة

2. حدد شكل الورقة:
   - ورقة إجابة كاملة
   - بطاقات (٤ في الصفحة)
   - أسئلة + إجابة (A4 أفقي)

3. اضغط: تحديث المعاينة

4. اطبع: 🖨️ طباعة البطاقة
```

### 4️⃣ التصحيح الآلي

**التبويب: 📷 التصحيح**

```
1. ضع مفتاح الإجابة:
   - اختر الإجابة الصحيحة لكل سؤال
   - أو استخدم: تعبئة تجريبية

2. اختر صورة البطاقة:
   - من الكاميرا 📱
   - أو ملف ممسوح 📄

3. حدد الأركان الأربعة (المربعات السوداء)
   أو اضغط: ✨ كشف تلقائي

4. اضغط: ✅ صحّح البطاقة

5. تحقق من النتائج وأضف اسم الطالب

6. اضغط: 💾 حفظ في النتائج
```

### 5️⃣ عرض النتائج

**التبويب: 📊 النتائج**

```
- عرض جميع النتائج المصححة
- تعديل الأسماء
- تصدير إلى CSV:
  → اضغط: ⬇️ تصدير Excel/CSV
```

## 🔒 الأمان والخصوصية

### حماية البيانات

1. **مفاتيح Firebase:**
   - ✅ محفوظة في `config.js` (غير مشاركة)
   - ✅ مضافة في `.gitignore`
   - ✅ لا تُرسل إلى GitHub

2. **البيانات المحلية:**
   - ✅ تُخزن في متغيرات JavaScript (غير مشفرة)
   - ✅ يمكن تصديرها ك CSV
   - ✅ لا تُرسل تلقائياً للسحابة

3. **الصور:**
   - ✅ تُعالج محلياً فقط
   - ✅ لا تُحفظ إلا كنتائج (أرقام)
   - ✅ لا تُرسل للخادم

### متطلبات Firebase الأمنية

```javascript
// مثال قواعد Firestore الآمنة:
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if request.auth != null;
    }
  }
}
```

## ⚙️ الإعدادات المتقدمة

### تخصيص عدد الأسئلة
في HTML، ابحث عن:
```javascript
sections:[ { type:'mcq', count:40, opts:4 } ]
```
غيّر `count` إلى العدد المطلوب.

### تخصيص الألوان
في CSS، ابحث عن:
```css
:root {
  --teal: #0f766e;
  --ok: #1f9d55;
  --bad: #d64545;
  /* غيّر الألوان حسب رغبتك */
}
```

### تعطيل Firebase
إذا أردت العمل بدون سحابة:
```javascript
// لا تستدعِ: initFirebase()
// البرنامج سيعمل محلياً فقط
```

## 🐛 حل المشاكل الشائعة

### ❓ "كاميرا غير متاحة"
```
✓ تأكد من إذن الكاميرا في المتصفح
✓ استخدم HTTPS (أو localhost)
✓ جرب متصفح آخر
```

### ❓ "OpenCV لم يُحمّل"
```
✓ تأكد من الاتصال بالإنترنت
✓ استخدم Chrome أو Firefox
✓ امسح ذاكرة التخزين المؤقت
```

### ❓ "البيانات لم تُحفظ"
```
✓ تحقق من اتصال الإنترنت
✓ جرب الحفظ المحلي أولاً
✓ تأكد من إعدادات Firebase
```

### ❓ "الصورة لم تُقرأ بصحة"
```
✓ تأكد من وضوح الصورة
✓ اجعل المربعات السوداء مرئية
✓ حسّن الإضاءة
✓ جرب اليد بدل الكاميرا (scanner)
```
